python requests库

2开头 （请求成功）表示成功处理了请求的状态代码
3开头 （请求被重定向）表示要完成请求，需要进一步操作。 通常，这些状态代码用来重定向
4开头 （请求错误）这些状态代码表示请求可能出错，妨碍了服务器的处理
5开头（服务器错误）这些状态代码表示服务器在尝试处理请求时发生内部错误。 这些错误可能是服务器本身的错误，而不是请求出错
